## Project 1: [NAIBU — Interactive Game Design] 🎮🧠
**Bachelor's Thesis · La Sabana University · Meritorious Thesis (Distinction) · Grade 4.8/5.0**

NAIBU is a 2D video game designed to support mental health literacy through game mechanics. The project explores how interactive systems can create safe spaces for emotional reflection — using gameplay as a medium for empathy and self-awareness.

> *Selected for MAGI Expo 2024 · Exhibited at Future Forte's inaugural art showcase · Featured at Bunjil Place for one month.*

### Main Features
- Game design and animation built entirely in Unity with custom C# mechanics
- Narrative structure designed around emotional literacy frameworks
- Awarded Meritorious Thesis (Distinction) with a 100% scholarship recognition

## Project 2: [Blooming — Interactive Installation] 🌸🖥️
**Museum Exhibition · Bunjil Place**

Blooming is a generative interactive installation exhibited in a gallery setting for one month. The work invites audiences to engage with organic digital growth through physical interaction — exploring themes of emergence, nature, and presence.

> *Average dwell time significantly above gallery average for the quarter.*

### Main Features
- Real-time generative visuals that respond to audience interaction
- Full museum exhibition lifecycle: concept → development → installation
- Multisensory experience combining sound design and motion graphics

- Non-linear structure that changes with user interaction

## Project 4: [Chasqui — Game Jam Entry] 🏃‍♀️🎮
**Women Global Game Jam 2020**

Chasqui is a game developed in 48 hours during the Women Global Game Jam 2020 — one of 10 games selected from the event. Inspired by Andean mythology, the game follows a Chasqui (messenger) on a sprint through ancient landscapes.

### Main Features
- Designed, developed, and shipped in a 48-hour game jam
- Selected in the Top 10 Games at Women Global Game Jam 2020
- Cultural storytelling through Andean mythology and visual design
